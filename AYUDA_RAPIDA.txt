🚀 COMPILAR Y EJECUTAR (UN SOLO COMANDO):

    ./run.sh

   Eso es todo! El script automáticamente:
   - Detecta tu sistema (Linux/macOS)
   - Compila el programa
   - Te pregunta si quieres ejecutarlo


📌 OTROS COMANDOS ÚTILES:

   Solo compilar (sin ejecutar):
     ./compile.sh

   Ejecutar después de compilar:
     ./matrix_mult

   Limpiar archivos compilados:
     make clean
     (o eliminar manualmente: rm -rf obj matrix_mult)


❓ SI ALGO NO FUNCIONA:

   1. ¿No tienes g++?
      En ZorinOS/Ubuntu ejecuta:
        sudo apt-get update
        sudo apt-get install build-essential

   2. ¿El script no es ejecutable?
      chmod +x run.sh
      chmod +x compile.sh

   3. ¿Error de permisos?
      Asegúrate de estar en la carpeta correcta:
        cd "/home/zandres/Escritorio/Septimo/Programacion Paralela/31-oct-2025"


🎯 USO DEL PROGRAMA:

   Una vez ejecutado, verás un menú:

   1 → Ver información del hardware de tu PC
   2 → Ejecutar secuencial (1 proceso)
   3 → Ejecutar paralelo (máximo de procesos)
   4 → Benchmark completo (prueba 1, 2, 4, 8... procesos)
   5 → Cambiar tamaño de matriz
   6 → Ver configuración actual
   7 → Salir

   RECOMENDADO: Empieza con opción 4 (Benchmark completo)
